## When to use this skill:

- When creating new database model or entity classes
- When modifying existing model definitions or adding fields
- When working on files such as models.py, models/*.py, schema.py, entities.py, or ORM configuration files
- When defining foreign key relationships or cascade behaviors
- When adding database constraints (NOT NULL, UNIQUE, CHECK)
- When choosing appropriate data types for new columns
- When adding indexes to model fields
- When setting up timestamps (created_at, updated_at) on models
- When balancing normalization vs. query performance in model design
