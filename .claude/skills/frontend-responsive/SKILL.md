Use this skill when building or modifying layouts that need to work across mobile, tablet, and desktop screen sizes. Use when working on files like *.css, *.scss, *.jsx, *.tsx, *.vue, *.html, or any file that defines layout, breakpoints, or media queries. Use when adding responsive breakpoints, converting pixel values to rem/em, optimizing images for mobile, or ensuring readable typography across screen sizes.
---

## When to use this skill:

- When building new page layouts or modifying existing ones for multiple screen sizes
- When working on files with media queries, breakpoints, or responsive utility classes
- When implementing mobile-first CSS or progressive enhancement for larger screens
- When converting fixed pixel values to relative units (rem, em, %)
- When adding or adjusting responsive breakpoints
- When sizing touch targets for mobile devices (minimum 44x44px)
- When optimizing images and assets for mobile network conditions
- When ensuring typography remains readable across all breakpoints
- When prioritizing content layout for smaller screens
- When testing or verifying UI across mobile, tablet, and desktop views
