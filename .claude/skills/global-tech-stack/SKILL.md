## When to use this skill:

- When selecting or recommending libraries, frameworks, or tools for new features
- When working on dependency configuration files (pyproject.toml, package.json, Gemfile, requirements.txt)
- When making architectural decisions about which technologies to use
- When setting up new services, databases, or infrastructure components
- When configuring testing frameworks, linters, or formatters
- When setting up deployment, hosting, or CI/CD pipelines
- When integrating third-party services (authentication, email, monitoring)
- When ensuring new additions align with the project's established stack
