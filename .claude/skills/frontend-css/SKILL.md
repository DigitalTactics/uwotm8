## When to use this skill:

- When creating or editing stylesheets (*.css, *.scss, *.less, *.module.css)
- When working with Tailwind CSS utility classes or tailwind.config.js
- When applying BEM naming conventions or CSS module patterns
- When defining or updating design tokens (colors, spacing, typography)
- When adding styles to components or templates
- When overriding or extending framework styles
- When optimizing CSS for production (purging unused styles, tree-shaking)
- When maintaining consistency across the project's visual design system
- When choosing between utility classes and custom CSS for new styles
