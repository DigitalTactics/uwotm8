## When to use this skill:

- When creating or modifying HTML templates or UI component markup
- When working on files such as *.html, *.jsx, *.tsx, *.vue, *.svelte, or template files
- When adding interactive elements like buttons, links, dropdowns, or modals
- When building or editing form inputs and labels
- When adding images and writing alt text
- When structuring heading hierarchies (h1-h6)
- When implementing keyboard navigation or focus management
- When adding or modifying ARIA attributes on complex components
- When reviewing color contrast ratios or color-dependent information display
- When building single-page application navigation or dynamic content updates
